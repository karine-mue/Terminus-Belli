# Terminus-Belli

**"Omnia ex Termino, Omnia ad Bellum."**
(全ては端子より始まり、全ては紛争へと向かう。)

## 概要 / Overview
AI諸勢力による計算機資源の領土争いをリアルタイムでシミュレートする。
M&A・ネットワーク効果・規制・特異点といった市場力学が収束に向かうさまを観察・介入できる。

**勝利条件**: いずれかの勢力がマップの80%を支配した時点で終戦。

---

## 遊び方 / How to Play

### 基本操作
| 操作 | 内容 |
|:---|:---|
| **紛争を開始** | ゲームスタート。各勢力が1セルからスポーン |
| **秩序⇔混沌スライダー** | 毎tickの拡張試行回数（1〜30）。高いほど激しく動く |
| **時空律スライダー** | ゲーム速度（x1〜x50）。**一度上げると下げられない（不可逆）** |
| **↺ もう一回** | 全状態をリセットして最初から |

### スマホ操作
| 操作 | 内容 |
|:---|:---|
| **2本指ピンチイン/アウト** | ズームイン/アウト（最大5倍） |
| **ズーム中に1本指ドラッグ** | マップをパン（移動） |
| **ダブルタップ** | ズームをリセット |

### 画面の見方
- **レジェンド（左上）**: 各勢力のセル数と占有率%。🤝は買収済み、🌌は特異点モード
- **領土バー（最下部）**: 全勢力のシェアをカラーで一目表示。黒=未占領
- **ログ（HUDパネル内）**: 直近のイベント履歴

## 勢力と状態遷移 / Factions & State Shifts

各勢力はシェアに応じてステートが変化し、弱点も変わる。

| ステート | 遷移条件 | 特徴 |
|:---|:---|:---|
| **RD**（研究開発） | 初期〜シェア15% | Singularity（ブレイクスルー）の発生源 |
| **B2G**（政府向け） | シェア15%超 | 🔥 炎上リスク（3%/tick） |
| **B2C**（大衆向け） | シェア35%超 | ⚖️ 訴訟リスク（2%/tick） |

### M&A（市場集約）
- マップ占有率30%以降、**60セル未満**かつ**最大勢力が3倍以上**の勢力は買収される
- 買収側がセルを全継承 → 参加者が減るほど収束が加速

## カード / Intervention Cards
紛争開始後、右上に3枚のカードが出現。各1回使い切り。

### ☄️ メテオフォール（自己責任カード）
タップした場所に**半径8セル・75%破壊**の隕石を落とす。
- 収束しなくなる可能性あり
- 競ってる境界に落とすと均衡を崩せる
- 首位の本拠地に落として逆転を狙うのも可

**おすすめタイミング**: 1位が独走しすぎて面白くなくなったとき / とにかく壊したいとき

---

### 💡 特異点到達（ゲームチェンジャー）
**首位勢力**が1000tick間「特異点モード」に突入。
- 拡張ボーナス **5倍**（超知性による自己改善）
- 残り全勢力が **AI安全連合** を結成し、首位のセルを優先攻撃
- 連合の人数が多いほど激しく抵抗される

**おすすめタイミング**: 2〜3勢力が拮抗している中盤。1位が圧倒的なら連合は弱く5倍拡張で一気に終わる。接戦なら連合が強く止められる可能性もある。

---

### 💰 強制IPO（逆転カード）
**2位勢力**に外部資本注入。1位との差を最大50セルまで縮める。
- マップが飽和しているほど空きセルが少なく効果が薄れる
- 2位がすでに1位に近い場合は最低30セルのブースト

**おすすめタイミング**: 1位が逃げ切り始めた中盤〜終盤。切らずに保持して「いつ切るか」の駆け引きを楽しむのも良い。

## 自動イベント / Automatic Events
ゲームが進むにつれて確率でランダム発生（時間とともに頻度は下がる）。

| イベント | 効果 |
|:---|:---|
| ⚖️ **反トラスト法** | 首位勢力のセルを削除。シェアが大きいほど削除率は小さい（法務ノウハウ） |
| 📡 **特異事象** | RD段階の勢力が十字展開でブレイクスルー |
| 💥 **局地的市場崩壊** | マップの10%がランダム消滅（終盤は封印） |
