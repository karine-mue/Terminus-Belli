# Terminus-Belli

**"Omnia ex Termino, Omnia ad Bellum."**
(全ては端子より始まり、全ては紛争へと向かう。)

## 概要 / Overview
本系（System）は、フローチャートにおける「端子（Terminal）」のクリックをトリガーとし、計算機資源の境界線を巡るAI諸勢力の侵略的推論（Invasive Inference）を可視化するシミュレーターである。

楕円の境界線はもはや「開始」を意味しない。それは、不可逆なエントロピーの増大、すなわち「紛争」の特異点である。
